Key Architectural Decisions

Source of Truth

Neo4j — all edges carry native relationship properties (conditions, techniques, strength). Obsidian vault is a generated mirror.

Schema-Driven

_meta/schema.yaml defines valid types, edges, attributes. Pipeline reads it at runtime to generate prompts and validate output.

Two-Pass Extraction

Pass 1 (concepts) per-section → merge → Pass 2 (claims) per-section grounded against merged concepts. Pass 3 enriches edges.

Dedup Cascade

Embedding pre-filter → string normalization → LLM confirmation. Each stage reduces work for the next.
