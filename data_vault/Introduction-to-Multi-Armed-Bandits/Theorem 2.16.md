# Theorem 2.16

For each problem instance $\mathcal{I}$ and any algorithm that satisfies (32), - (a) the bound (33) holds with $C_{\mathcal{I}}=\sum_{a:\;\Delta(a)>0}\;\frac{\mu^{*}(1-\mu^{*})}{\Delta(a)}.$ - (b) for each $\epsilon>0$, the bound (33) holds with $C_{\mathcal{I}}=C^{0}_{\mathcal{I}}-\epsilon$, where $C^{0}_{\mathcal{I}}=\sum_{a:\;\Delta(a)>0}\;\frac{\Delta(a)}{\mathtt{KL}(\mu(a),\,\mu^{*})}-\epsilon.$
