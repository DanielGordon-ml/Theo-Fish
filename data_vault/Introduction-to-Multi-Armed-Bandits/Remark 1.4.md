# Remark 1.4

With this approach, one does not need to worry about probability in the rest of the analysis. Indeed, the probability has been taken care of by defining the clean event and observing that (2) holds therein. We do not need to worry about the bad event, either: essentially, because its probability is so tiny. We will use this “clean event” approach in many other proofs, to help simplify the technical details. This simplicity usually comes at the cost of slightly larger constants in $O()$, compared to more careful arguments which explicitly track low-probability events throughout the proof.
