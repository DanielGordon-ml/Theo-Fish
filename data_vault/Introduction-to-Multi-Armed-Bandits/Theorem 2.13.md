# Theorem 2.13

No algorithm can achieve regret $\operatornamewithlimits{\mathbb{E}}[R(t)]=o(c_{\mathcal{I}}\;\log t)$ for all problem instances $\mathcal{I}$, where the “constant” $c_{\mathcal{I}}$ can depend on the problem instance but not on the time $t$.
