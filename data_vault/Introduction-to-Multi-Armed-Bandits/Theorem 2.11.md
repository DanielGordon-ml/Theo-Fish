# Theorem 2.11

Consider any algorithm which satisfies non-adaptive exploration. Fix time horizon $T$ and the number of arms $K<T$. Then there exists a problem instance such that $\operatornamewithlimits{\mathbb{E}}[R(T)]\geq\Omega\left(\,T^{2/3}\cdot K^{1/3}\,\right)$.
