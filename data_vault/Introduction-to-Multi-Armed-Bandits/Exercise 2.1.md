# Exercise 2.1

Prove Theorems 2.11 and 2.12. Specifically, consider an algorithm which satisfies non-adaptive exploration. Let $N$ be the number of exploration rounds. Then: - (a) there is a problem instance such that $\operatornamewithlimits{\mathbb{E}}[R(T)]\geq\Omega(\;T\cdot\sqrt{K/\operatornamewithlimits{\mathbb{E}}[N]}\;)$. - (b) for each problem instance, randomly permuting the arms yields $\operatornamewithlimits{\mathbb{E}}[R(T)]\geq\operatornamewithlimits{\mathbb{E}}[N]\cdot\tfrac{1}{K}\sum_{a}\Delta(a)$. - (c) use parts (a,b) to derive Theorems 2.11 and 2.12.
