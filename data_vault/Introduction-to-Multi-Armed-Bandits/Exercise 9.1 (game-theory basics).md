# Exercise 9.1 (game-theory basics)

- (a) Prove that a distribution $p\in\Delta_{\mathtt{rows}}$ satisfies Eq.\xa0(120) if and only if it is a minimax strategy.
Hint: $M(p^{*},q)\leq f(p^{*})$ if $p^{*}$ is a minimax strategy; $f(p)\leq v^{*}$ if $p$ satisfies (120).
- (b) Prove that $p\in\Delta_{\mathtt{rows}}$ and $q\in\Delta_{\mathtt{cols}}$ form a Nash equilibrium if and only if $p$ is a minimax strategy and $q$ is a maximin strategy.
