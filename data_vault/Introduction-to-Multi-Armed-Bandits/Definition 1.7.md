# Definition 1.7

A round $t$ is an *exploration round* if the data $\left(\,a_{t},r_{t}\,\right)$ from this round is used by the algorithm in the future rounds. A deterministic algorithm satisfies *non-adaptive exploration* if the set of all exploration rounds and the choice of arms therein is fixed before round $1$. A randomized algorithm satisfies *non-adaptive exploration* if it does so for each realization of its random seed.
