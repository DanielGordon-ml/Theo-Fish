# Remark 11.16

The theorem focuses on the posterior gap $G$ given $N_{0}$ initial samples from arm $1$. The theorem requires parameters $\epsilon>0$ and $N_{0}$ to satisfy some condition that depends only on the prior. Such parameters exist if and only if $\Pr[G>0]>0$ for some $N_{0}$ (for precisely the same reason as in Remark 11.11). The latter condition is in fact necessary, as we will see in Section 65.
