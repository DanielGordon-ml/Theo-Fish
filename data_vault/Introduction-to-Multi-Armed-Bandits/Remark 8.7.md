# Remark 8.7

Algorithm\xa03 is modular in two ways: it can take an arbitrary classification oracle, and it can use any other unbiased estimator instead of Eq.\xa0(110). In particular, the proof below only uses the fact that Eq.\xa0(110) is an unbiased estimator with $\tilde{r}_{t}(a)\leq K$.
