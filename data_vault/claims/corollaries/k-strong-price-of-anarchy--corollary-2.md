# Corollary 2

For the class of resource allocation problems 𝒢_n × {w}, if the value of (Qζ) satisfies ρ_ζ* = Q*(n, w, k), then SPoA_k*(𝒢_n, w) = 1 / Q*(n, w, k) is a tight bound and a solution ũ_ζ to (Qζ) is an optimal utility rule.
