# Lemma 3 (Stochastic Stability Characterization via Minimum Resistance Trees)

A state z = [ā, ū] in C⁰ is stochastically stable if and only if ā ∈ arg min_{a* ∈ A} { c(|C⁰| - 1) + ∑_{i ∈ N} (1 - U_i(a*)) }, equivalently ā ∈ arg max_{a* ∈ A} { ∑_{i ∈ N} U_i(a*) }.
