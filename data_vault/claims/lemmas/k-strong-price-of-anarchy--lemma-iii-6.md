# Lemma III.6

Consider the class of resource allocation problems 𝒢_n with local welfare rule w. For a given label parameter vector θ and corresponding coalition set C_ζ, the coalitional smoothness parameters λ, μ can be computed by solving the smoothness parameter optimization problem P_ζ, which is a tractable linear program.
