# Lemma 3.1 (Key Lemma)

Let M be an ergodic Markov chain with state space [n], stationary distribution π, and spectral expansion λ = λ(M). For any diagonal matrix P with entries in [0,1], we have ‖PM‖_π ≤ (1 + λ)/2, and ‖P^⊥M‖_π ≤ (1 + λ)/2, where P^⊥ = I − P.
