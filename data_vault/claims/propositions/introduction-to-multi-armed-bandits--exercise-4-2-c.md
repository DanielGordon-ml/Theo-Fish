# Exercise 4.2(c)

For an arbitrary metric space $(X,\mathcal{D})$ and an arbitrary $\epsilon$-net $\mathcal{N}$ therein, for any algorithm there is a problem instance on $(X,\mathcal{D})$ such that $\operatornamewithlimits{\mathbb{E}}[R(T)]\geq\Omega\left(\min\left(\epsilon T,\,|\mathcal{N}|/\epsilon\right)\right)$ for any time horizon $T$.
