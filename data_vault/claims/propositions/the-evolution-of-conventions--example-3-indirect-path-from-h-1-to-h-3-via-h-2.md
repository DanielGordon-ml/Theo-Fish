# Example 3 — Indirect path from h_1 to h_3 via h_2

The direct route from h_1 to h_3 requires some player to make [(5/8)k] mistaken choices of strategy 3, since otherwise strategy 1 or 2 is a better reply. But there is a path of less resistance, namely, go first to h_2 and then to h_3. The total resistance of this path is [(1/8)k] + [(2/5)k], which is approximately (21/40)k when k is large.
