# Proposition III.2

Each resource allocation problem (G,w) ∈ 𝒢_n × {w} is (λ,μ)-k-coalitionally smooth with λ_ζ = 1/ν_ζ^⋆ and μ_ζ = ρ_ζ^⋆/ν_ζ^⋆ - 1, where (ρ_ζ^⋆, ν_ζ^⋆) is a solution to the linear program (Pζ).
