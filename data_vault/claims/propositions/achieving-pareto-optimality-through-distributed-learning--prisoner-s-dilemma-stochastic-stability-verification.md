# Prisoner's Dilemma Stochastic Stability Verification

For the two-player prisoner's dilemma game with payoffs U_1(A,A) = U_2(A,A) = 3/4, U_1(A,B) = U_2(B,A) = 0, U_1(B,A) = U_2(A,B) = 4/5, U_1(B,B) = U_2(B,B) = 1/3, the action profile (A,A) is the unique stochastically stable state under the proposed algorithm with c = n = 2. The recurrence classes of the unperturbed process are (AA, AB, BA, BB, ∅), where agents are content for the four listed action profiles and ∅ corresponds to the scenario where both agents are discontent. The stochastic potential of AA is minimum among all states.
