# GoT Algorithm (fully distributed multi-player multi-armed bandit algorithm)

We present a distributed algorithm and prove that it achieves an expected sum of regrets of near-O(log^2 T). This is the first algorithm to achieve a poly-logarithmic regret in this fully distributed scenario.
