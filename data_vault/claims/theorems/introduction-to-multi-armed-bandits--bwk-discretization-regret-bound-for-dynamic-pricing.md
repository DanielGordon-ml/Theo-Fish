# BwK discretization regret bound for dynamic pricing

With the discretization approach, choosing a finite subset P ⊂ [0,1] of prices and running a generic BwK algorithm with action set P, Badanidiyuru et al. (2013, 2018) achieve regret bound Õ(B^{2/3}) against OPT.
