# Regret bound for linear contextual BwK (Eq. 155)

For linear contextual BwK, Agrawal and Devanur (2016) achieve regret bound OPT - E[REW] ≤ Õ(m√T)(1 + OPT/B) in the regime B > mT^{3/4}.
