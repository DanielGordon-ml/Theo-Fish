# Theorem 2 (Young/Kandori-Mailath-Rob framework)

Let P^0 be a stationary Markov chain on a finite state space X with recurrent communication classes X_1, X_2, ..., X_J. Let P^ε be a regular perturbation of P^0. Then: (i) P^ε has a unique stationary distribution μ^ε for each ε ∈ (0, ε*]. (ii) lim_{ε→0} μ^ε = μ^0 exists, and μ^0 is a stationary distribution of P^0. (iii) A state x is stochastically stable (μ^0(x) > 0) if and only if x is contained in some recurrent communication class X_j that has minimum stochastic potential.
