# Theorem 2.16(b)

For each problem instance I and any algorithm that satisfies (32), for each ε > 0, the bound (33) holds with C_I = C^0_I - ε, where C^0_I = ∑_{a: Δ(a)>0} Δ(a)/KL(μ(a), μ*).
