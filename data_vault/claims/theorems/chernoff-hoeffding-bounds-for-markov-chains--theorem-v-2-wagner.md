# Theorem V.2 (Wagner)

Under the same setting as Theorem V.1 (but with a single fixed weight function f for all steps), for any δ ∈ (0,1], Pr[X ≥ (1+δ)μ] ≤ n · exp(−μδ²γ/(8 ln(2e/γ))).
