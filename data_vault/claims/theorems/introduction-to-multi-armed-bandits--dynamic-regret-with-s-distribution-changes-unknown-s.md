# Dynamic regret with S distribution changes (unknown S)

The regret bound for unknown $S$ can be improved to $\operatornamewithlimits{\mathbb{E}}[R^*(T)] = \tilde{O}(\sqrt{SKT})$, matching the optimal regret rate for known $S$, using more advanced algorithms (Auer et al., 2019; Chen et al., 2019).
