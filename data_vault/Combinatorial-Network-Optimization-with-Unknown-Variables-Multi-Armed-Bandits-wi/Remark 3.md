# Remark 3

Theorem 2 also holds for random variables $X _ { i } , 1 \le i \le N$ that are not i.i.d. over time, but with the only weaker assumption that $E [ X _ { i } ( t ) | X _ { i } ( 1 ) , \dots , X _ { i } ( t - 1 ) ] =$ $\begin{array} { r } { \theta _ { i } , \forall 1 \le i \le N . } \end{array}$ . This is because the Chernoff-Hoeffding bound only needs a weak assumption $E [ X _ { i } ( t ) | X _ { i } ( 1 ) , \dots , X _ { i } ( t -$ $1 ) ] = \theta _ { i } , \forall 1 \leq i \leq N$ .
