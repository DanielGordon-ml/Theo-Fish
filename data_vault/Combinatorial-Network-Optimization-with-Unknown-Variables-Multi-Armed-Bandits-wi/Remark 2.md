# Remark 2

We have presented $\mathcal { F }$ as a finite set in our problem formation. We note that the LLR policy we have described and its analysis actually also work with a more general formulation when $\mathcal { F }$ is an infinite set with the following additional constraints: the maximization problem in (4) always has at least one solution; $\Delta _ { \mathrm { m i n } }$ exists; $a _ { i }$ is bounded. With the above constraints, Algorithm 1 will work the same and the conclusion and all the details of the proof of Theorem 2 can remain the same.
